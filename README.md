# Tarjeta de 15 — Luana

Invitación digital (web mobile-first) estilo *pink disco glam*. Se ve desde el celu y se comparte por WhatsApp.

## Archivos
- `index.html` — la página completa (HTML + CSS + JS, todo adentro)
- `assets/` — imágenes de fondo (`bg-hero.png`, `bg-2/3/4.png`)

## Cómo editar
Todos los datos (nombre, fecha, lugar, links, precios, fondos, música) están en el objeto **`CONFIG`** al final de `index.html`. No hace falta tocar nada más.

Para una foto real: en un `<div class="photo">` agregá `style="background-image:url('mi-foto.jpg')"`.
